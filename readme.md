## 2. vue-element-form-generator用途
* 使用JSON配置的方式生成表单，简化开发
* 使用统一的方式进行页面布局，使布局统一

## 3. 支持组件
* ElementUI的组件: 
    * Switch（开关）
    * Input（文本框）
    * Button（按钮）
* 添加的组件
    * Fill（填充整行）
