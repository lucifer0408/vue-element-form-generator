# vue-element-form-generator

## 1. 工程描述

> vue-element-form-generator的示例工程

## 2. vue-element-form-generator用途
* 使用JSON配置的方式生成表单，简化开发
* 使用统一的方式进行页面布局，使布局统一

## 3. 支持组件
* ElementUI的组件: 
    * Switch（开关）
    * Input（文本框）
    * Button（按钮）
    * Number（数字）
    * Radio（单选框）
    * Checkbox（复选框）
    * Select（下拉框）
* 添加的自定义组件
    * Fill（填充占位）
